# Bangalore Job Alert Bot

This Telegram bot sends daily job alerts for:
- Finance jobs
- Analytics jobs
- Data-related and high-paying roles

### Features
- Location: Bangalore
- Sources: Company career pages, LinkedIn, Naukri (via keywords)
- Schedule: Everyday at 10 PM
- Notifications via Telegram

### How to run
1. Install dependencies
2. Add Telegram Bot Token and Chat ID
3. Run the script
